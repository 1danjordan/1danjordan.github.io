Visualising Collatz

In previous posts I used the Collatz conjecture as an example for writing:

  • safe recursion and memoisation
  • tidy summary functions

Now I’m going to build further on this example and finally visualise the hailstone sequence. I’m going to use ggplot and Plotly’s R package plotly to create some really nice interactive charts. The plotly package is the easiest way to turning your graphs from static to interactive, and the ggplot to plotly conversion is perfect in nearly all the times I’ve used it.

These functions are explained in detail in my previous posts.

library(purrr)

collatz <- function(x) {
  .f <- function(x) {
    n <- tail(x, 1)
    if (n == 1) return(x)
    else if(n %% 2 == 0) Recall(c(x, n / 2))
    else Recall(c(x, 3 * n + 1))
  }
  
  structure(lapply(x, .f), class = "collatz")
}

summary.collatz <- function(object) {
  tibble::tibble(
    n        = map_dbl(object, ~ head(.x, 1)),
    sequence = object,
    steps    = map_int(object, length),
    max      = map_dbl(object, max)
  )
}

The Collatz conjecture wiki has three graphs that capture some of the beauty of the hailstone sequence. Let’s recreate them with ggplot. In the right margin, you can see each of the graphs that I am recreating.

Stopping Time for Each Initial Value

Graph 1: Collatz Stopping Time 

library(ggplot2)
library(plotly, warn.conflicts = FALSE)

p1 <- collatz(1:10000) %>% 
  summary %>%
  ggplot(aes(n, steps)) +
  geom_point(alpha = 0.5, colour = "blue", size = 0.45) +
  theme_minimal() +
  guides(colour = "none") +
  labs(
    title = "Stopping Time",
    subitle = "for intial values 1:10000", 
    x = "Stopping Time", 
    y = "Initial Value"
  )

ggplotly(p1)

Histogram of Total Stopping Times

Graph 2: Histogram of Total Stopping Times 

p2 <- collatz(1:10000) %>%
  summary %>%
  ggplot(aes(steps)) +
  geom_histogram(binwidth = 2, fill = "#265EF1") +
  theme_minimal() +
  labs(
    title = "Histogram of Total Stopping Times",
    subtitle = "For initial values 1:10000",
    x = NULL, 
    y = NULL
  )

ggplotly(p2)

Maximum Value Reached by the Sequence

Graph 3: Maximum Reached by Sequence 

p3 <- collatz(1:10000) %>% 
  summary %>% 
  ggplot(aes(n, max)) +
  geom_point(colour = "blue", size = 0.4, alpha = 0.35, na.rm = TRUE) +
  scale_y_continuous(limits = c(0, 100000)) +
  theme_minimal() +
  labs(
    title = "Maximum Reached by the Sequence",
    x = "Initial Value",
    y = "Maximum"
  )

ggplotly(p3)

So there it is - using simple composable pieces; a short function, a tidy summary method and some basic plotting we have a beautifully simple way of computing and plotting the hailstone sequence.
