A small R package on Irish demography

 Mar 1, 2019 R package

I’ve made a new R package called irishbabynames. Install it with:

remotes::install_github("1danjordan/irishbabynames")

This package contains three datasets:

  • babynames - count of each baby name in Ireland from 1964
  • migration - migration statistics by gender and age group from 1987
  • population - population statistics by sex and age from 1926

This is only available on Github and I’ve no plans on publishing it on CRAN for the moment. I’ve wanted to make a data package for a while now, and this week the Irish CSO had a big press release about the statistics for 2018.

An Evenings Work

It only took an hour or so to scrape the data from the CSO website and pull the package together. With Rstudio and packages like usethis, creating a package couldn’t be easier. Oddly enough the Irish CSO publish data in a format called PC-AXIS, which I had never heard of until yesterday. This was the only challenge really but luckily for me, as always CRAN provides. the pxR made it simple to read the file in and transform it into a data frame.

I gathered this particular datasets into a package because I think they capture a lot of Ireland’s story in the second half of the 20th century. We might be able to do some nice analysis with it - maybe model the probability of emigrating Ireland given a particular name, sex and age? If nothing else, I’ve made a few nice plots with it.

This is pretty much an exact rip-off of Hadley Wickham’s babynames package. But let’s call it inspiration instead. As they say, imitation is the sincerest form of flattery.
